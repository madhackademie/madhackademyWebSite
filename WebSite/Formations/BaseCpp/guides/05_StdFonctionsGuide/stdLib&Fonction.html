Librairie Standard & Fonctions C++

GameDevReady

Objectif du Module

Avant de plonger dans les détails, prenons un moment pour parler de la STL (Standard Template Library). Couramment appeler "STD" et vous comprendrez vite pourquoi. Nous avons déjà utilisé un outil de la std avec printf qui vient de la bibliothèque cstdio.

La std correspond à "librairie standard". Et elle contient plein d'outils pratiques pour nous simplifier la vie et nous éviter de coder tout par nous-mêmes. Vous serez amenés à l'utiliser tout au long de votre carrière de programmeur C++.

Dans ce module, nous allons découvrir trois concepts fondamentaux qui vont révolutionner votre façon de coder :

  • La bibliothèque string : pour manipuler du texte facilement
  • La bibliothèque iostream : pour communiquer avec l'utilisateur (entrées/sorties)
  • Les fonctions : pour organiser et réutiliser votre code

Ce que l'élève saura faire à la fin

Créer et utiliser des fonctions simples et avec paramètres / retour, manipuler des chaînes de caractères avec string, et interagir avec l'utilisateur via cin et cout.

Notions techniques abordées : Fonctions (déclaration, paramètres, retour), std::string, std::cout, std::cin, flux d'entrée/sortie.

La Bibliothèque String : Manipuler du Texte

Pourquoi utiliser std::string ?

Jusqu'à présent, nous avons utilisé que les options de texte présentes dans les fonctions fprint et DrawText(textformat()), mais si vous voulez créer une variable de type text en C++, vous devrez utiliser la bibliothèque string. Car l'ancienne méthode héritée du C char* est limitée et compliquée.
Voire inutilisable pour les dictionnaires (tableaux clés/valeurs vus ultérieurement).
La bibliothèque string de la STL nous offre un type moderne et facile à utiliser : std::string.

Avantages de std::string :

  • ✅ Taille dynamique (pas besoin de spécifier la longueur)
  • ✅ Facile à modifier (ajouter, supprimer, remplacer du texte)
  • ✅ Gestion automatique de la mémoire
  • ✅ Nombreuses fonctions utiles intégrées

Pour utiliser string, il faut inclure la bibliothèque :

#include <string>

Création et utilisation basique

#include <cstdio>
#include <string>

int main() 
{
    // Créer une string
    std::string nomHeros = "Grenouille";
    std::string message = "Bienvenue dans le jeu !";
    
    // Afficher avec printf (conversion nécessaire)
    printf("Héros : %s\n", nomHeros.c_str());
    printf("Message : %s\n", message.c_str());
    
    return 0;
}

⚠️ Note importante :

printf ne peut pas afficher directement une std::string. Il faut utiliser .c_str() pour la convertir en const char*.

C'est pour ça que nous allons bientôt découvrir cout, qui fonctionne parfaitement avec std::string !

Opérations courantes avec string

#include <string>
#include <cstdio>

int main() 
{
    std::string prenom = "Mario";
    std::string nom = "Bros";
    
    // Concaténation (assemblage de textes)
    std::string nomComplet = prenom + " " + nom;
    printf("Nom complet : %s\n", nomComplet.c_str());
    
    // Ajouter du texte à la fin
    std::string message = "Bonjour";
    message += " le monde";
    message += "!";
    printf("%s\n", message.c_str());  // Affiche : Bonjour le monde!
    
    // Longueur du texte
    std::string texte = "Frogger";
    int longueur = texte.length();  // ou texte.size()
    printf("Le mot '%s' contient %d lettres\n", texte.c_str(), longueur);
    
    // Accéder à un caractère spécifique
    char premiereLettre = texte[0];  // 'F'
    char derniereLettre = texte[longueur - 1];  // 'r'
    printf("Première lettre : %c\n", premiereLettre);
    printf("Dernière lettre : %c\n", derniereLettre);
    
    // Vérifier si la string est vide
    std::string vide = "";
    if (vide.empty()) {
        printf("La chaîne est vide\n");
    }
    
    return 0;
}

Fonctions utiles de string

Fonction Description Exemple
.length() Taille du texte texte.length()
.empty() Vérifie si vide if (texte.empty())
.clear() Vide la string texte.clear()
.c_str() Conversion en char* printf("%s", texte.c_str())
.append() Ajouter du texte texte.append(" fin")
.substr() Extraire une partie texte.substr(0, 5)

Exemple pratique : Gestion d'un nom de joueur

std::string nomJoueur = "Grenouille_Verte_123";

// Extraire seulement le prénom (jusqu'au premier _)
int position = nomJoueur.find('_');  // Trouve la position du _
std::string prenom = nomJoueur.substr(0, position);  // "Grenouille"

printf("Prénom : %s\n", prenom.c_str());

La Bibliothèque Iostream : Communiquer avec l'Utilisateur

Comprendre les flux d'entrée/sortie

En programmation, il existe deux grands flux de données :

  • Output (sortie) : Les données qui sortent de votre programme vers l'utilisateur
    → Exemple : Afficher du texte à l'écran
  • Input (entrée) : Les données qui entrent dans votre programme depuis l'utilisateur
    → Exemple : L'utilisateur tape son nom au clavier

Jusqu'à présent, nous avons utilisé printf() pour les sorties. Maintenant, découvrons iostream, une bibliothèque moderne C++ qui gère les entrées ET les sorties de manière élégante.

#include <iostream>

cout : Afficher du texte (Output)

std::cout (console output) est l'équivalent moderne de printf.

#include <iostream>
#include <string>

int main() 
{
    std::string nom = "Grenouille";
    int age = 5;
    float taille = 1.25f;
    
    // Afficher du texte avec cout
    std::cout << "Bienvenue dans le jeu !\n";
    
    // Afficher des variables (pas besoin de %s, %d, %f !)
    std::cout << "Nom : " << nom << "\n";
    std::cout << "Age : " << age << " ans\n";
    std::cout << "Taille : " << taille << " mètres\n";
    
    // Tout sur une ligne
    std::cout << "Joueur " << nom << " (" << age << " ans)\n";
    
    return 0;
}

Avantages de cout sur printf :

  • ✅ Pas besoin de se souvenir des spécificateurs (%d, %f, etc.)
  • ✅ Fonctionne nativement avec std::string
  • ✅ Syntaxe plus simple et moins d'erreurs possibles
  • ✅ Utilise l'opérateur << pour "envoyer" des données vers la sortie

Note sur \n vs std::endl :

  • \n : retour à la ligne simple (rapide)
  • std::endl : retour à la ligne + flush du buffer (plus lent mais garantit l'affichage immédiat)

Pour la plupart des cas, \n suffit amplement.

cin : Récupérer les entrées utilisateur (Input)

Voici la grande nouveauté : std::cin (console input) nous permet de demander des informations à l'utilisateur !

#include <iostream>
#include <string>

int main() 
{
    std::string nomJoueur;
    int ageJoueur;
    
    // Demander le nom
    std::cout << "Quel est votre nom ? ";
    std::cin >> nomJoueur;  // L'utilisateur tape son nom
    
    // Demander l'âge
    std::cout << "Quel est votre âge ? ";
    std::cin >> ageJoueur;  // L'utilisateur tape son âge
    
    // Afficher un message personnalisé
    std::cout << "\nBienvenue " << nomJoueur << " !\n";
    std::cout << "Vous avez " << ageJoueur << " ans.\n";
    
    return 0;
}

Comment ça fonctionne ?

L'opérateur >> extrait les données de l'entrée (clavier) et les stocke dans la variable.

Le programme attend que l'utilisateur tape quelque chose et appuie sur Entrée.

⚠️L'IDE se comporte de manière un peu différente car il attends quelque chose de vous. Mais comme moi, il n'est pas très fort en communication. Il faut voir dans la console, car elle vous pose une question (c'est d'ailleurs le comportement que nous attendions). Il faut avec la souris cliquer sur l'espace vide en-dessous de la question afin d'avoir le curseur de saisie. Puis donner votre réponse.

IMAGE : La console en attente de vos informations.

Et si pour une raison ou une autre vous vouliez arreter le debug ou programme sans répondre.

IMAGE : Comment arreter le programme manuellement.

Exemple complet : Création de personnage

#include <iostream>
#include <string>

int main() 
{
    std::string nomHeros;
    std::string classe;
    int niveau;
    
    std::cout << "=== CREATION DE PERSONNAGE ===\n\n";

    std::cin >> niveau;

Lire une ligne complète avec getline()

Par défaut, cin s'arrête au premier espace. Pour lire une phrase complète :

#include <iostream>
#include <string>

int main() 
{
    std::string phraseComplete;
    
    std::cout << "Entrez une phrase : ";
    std::getline(std::cin, phraseComplete);  // Lit toute la ligne
    
    std::cout << "Vous avez dit : " << phraseComplete << "\n";
    
    return 0;
}

⚠️ Piège courant avec getline :

Si vous utilisez cin avant getline, il faut vider le buffer : Il s'agit d'une histoire de gestion de mémoire résiduelle et de mémoire tempon. Je vous laisse effectuer vos propre recherche pour attiser votre curiosité si nécessaire. Mais n'oubliez pas " La vérité est une chose belle et terrible, et doit donc être traitée avec une grande précaution" 

int age;
std::string nom;

std::cin >> age;
std::cin.ignore();  // ⬅️ Important !
std::getline(std::cin, nom);

Tableau comparatif : printf/scanf vs cout/cin

Opération C (printf/scanf) C++ (cout/cin)
Afficher un entier printf("%d", x); cout << x;
Afficher un float printf("%.2f", y); cout << y;
Afficher une string printf("%s", s.c_str()); cout << s;
Lire un entier scanf("%d", &x); cin >> x;
Lire une string scanf("%s", buffer); cin >> s;

Les Fonctions : Réutiliser son Code

Qu'est-ce qu'une fonction ?

Imaginez que vous devez faire sauter notre grenouille plusieurs fois dans votre jeu. Sans fonction, vous devriez copier-coller le même code encore et encore. Avec une fonction, vous écrivez le code une seule fois et vous l'appelez autant de fois que nécessaire !

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. C'est comme une recette de cuisine : vous l'écrivez une fois, et vous pouvez la suivre autant de fois que vous voulez.

Vous connaissez déjà des fonctions !

  • printf() : affiche du texte dans la console
  • DrawText() : affiche du texte dans Raylib
  • InitWindow() : initialise une fenêtre Raylib

Toutes ces fonctions ont été créées par quelqu'un d'autre, et vous les utilisez simplement en les appelant avec leurs noms et leurs paramètres.

La fonction de base (sans paramètre, sans retour)

Commençons par le plus simple : une fonction qui ne prend rien et ne renvoie rien. Elle fait juste une action.

// Déclaration de la fonction (avant le main)
        void DireBonjour() 
        {
        printf("Bonjour, apprenti développeur !\n");
        printf("Prêt à coder aujourd'hui ?\n");
        }
        
        int main() 
        {
        // Appel de la fonction
        DireBonjour();
        DireBonjour();  // On peut l'appeler autant de fois qu'on veut !
        
        return 0;
        }

Décortiquons cette fonction :

  • void : signifie que la fonction ne renvoie aucune valeur
  • DireBonjour : le nom de la fonction (commence par une majuscule par convention)
  • () : les parenthèses vides indiquent qu'elle ne prend aucun paramètre
  • { ... } : le corps de la fonction, ce qu'elle fait réellement

La fonction avec paramètres

Maintenant, rendons notre fonction plus flexible en lui permettant de recevoir des informations.


                #include <iostream>
                #include <string>
                // Fonction qui salue une personne spécifique
                void SaluerJoueur(std::string nomJoueur, int niveau) 
                {
                std::cout <<"Salut "<< nomJoueur <<"\n";
                std::cout <<"Niveau actuel : "<< niveau << "\n";
                std::cout <<"Continue comme ça !\n\n";
                }
                
                int main() 
                {
                SaluerJoueur("Mario", 5);
                SaluerJoueur("Luigi", 3);
                SaluerJoueur("Grenouille", 42);
                
                return 0;
                }

Les paramètres :

Les paramètres sont comme des variables temporaires qui n'existent que dans la fonction. Ici, nomJoueur et niveau reçoivent les valeurs passées lors de l'appel.

  • const char* nomJoueur : une chaîne de caractères (texte)
  • int niveau : un nombre entier

La fonction avec retour de données

Parfois, on veut que notre fonction nous renvoie un résultat. C'est comme demander à quelqu'un de faire un calcul et de vous donner la réponse.

// Fonction qui calcule la puissance d'un saut
        float CalculerPuissanceSaut(float vitesse, float angle) 
        {
        float puissance = vitesse * angle * 0.5f;
        return puissance;  // On renvoie le résultat
        }
        
        int main() 
        {
        float resultat1 = CalculerPuissanceSaut(10.0f, 45.0f);
        printf("Puissance du saut : %.2f\n", resultat1);
        
        // On peut aussi utiliser le résultat directement
        if (CalculerPuissanceSaut(5.0f, 30.0f) > 50.0f) {
        printf("Super saut !\n");
        }
        
        return 0;
        }

⚠️ Important sur le return :

  • Le type de retour (float ici) doit correspondre au type de la valeur renvoyée
  • return termine immédiatement l'exécution de la fonction
  • Si le type de retour est void, pas besoin de return (ou juste return; sans valeur)

Exemple complet : Fonction de jeu


            #include <iostream>
            #include <string>
        
            // Fonction qui vérifie si le joueur a gagné
            bool VerifierVictoire(int score, int scoreRequis) 
            {
                if (score >= scoreRequis) {
                return true;  // Victoire !
                } else {
                return false; // Pas encore...
                }
            }
            
            // Fonction qui affiche le statut du joueur
            void AfficherStatut(std::string nom, int score, int vies) 
            {
                std::cout <<"=== STATUT JOUEUR ===\n";
                std::cout <<"Nom : " << nom << "\n";
                std::cout <<"Score : " << score << "\n";
                std::cout <<"Vies : " << vies << "\n";
                std::cout <<"=====================\n\n";
            }
            
            int main() 
            {
                std::string nomJoueur = "Grenouille";
                int scoreJoueur = 450;
                int viesJoueur = 3;
                int scoreVictoire = 500;
                
                AfficherStatut(nomJoueur, scoreJoueur, viesJoueur);
                
                if (VerifierVictoire(scoreJoueur, scoreVictoire)) {
                printf("Bravo ! Vous avez gagné !\n");
                } else {
                std::cout <<"Contine, encore "<< scoreVictoire - scoreJoueur << " points ! \n";
                printf("Continue, encore %d points !\n", scoreVictoire - scoreJoueur);
                }
                
                return 0;
            }

Résumé sur les fonctions

  • Sans paramètre, sans retour : void MaFonction() { ... }
  • Avec paramètres : void MaFonction(int x, float y) { ... }
  • Avec retour : int MaFonction() { return 42; }
  • Complet : float Calculer(int a, int b) { return a * b * 1.5f; }

Points Clés

  • Point clé 1 : Les fonctions permettent de réutiliser du code et d'organiser votre programme
  • Point clé 2 : std::string simplifie la manipulation de texte
  • Point clé 3 : cout et cin sont plus modernes et simples que printf/scanf

À retenir

Définitions

STD (Standard Library) : La bibliothèque standard du C++ qui contient de nombreux outils prêts à l'emploi : conteneurs (vector, array, etc.), algorithmes, gestion de texte, entrées/sorties, et bien plus. Tout ce qui commence par std:: provient de cette bibliothèque.

Fonction : Un bloc de code réutilisable qui effectue une tâche spécifique. Elle peut recevoir des paramètres (données en entrée) et renvoyer une valeur (résultat). Les fonctions permettent d'organiser le code et d'éviter les répétitions.

Paramètre : Variable locale à une fonction qui reçoit une valeur lors de l'appel de cette fonction. Les paramètres permettent de passer des informations à une fonction pour qu'elle puisse travailler avec.

Return (retour) : Instruction qui termine l'exécution d'une fonction et renvoie une valeur au code appelant. Le type de la valeur renvoyée doit correspondre au type de retour déclaré dans la signature de la fonction.

Buffer (tampon) : Zone de mémoire temporaire utilisée pour stocker des données avant leur traitement. Dans le contexte de cin, c'est l'endroit où les caractères tapés au clavier sont stockés en attendant d'être lus par le programme.
