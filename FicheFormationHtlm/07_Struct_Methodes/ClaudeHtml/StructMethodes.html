Struct & Methodes C++

GameDevReady

Objectif du Module

Jusqu'ici, une struct regroupe des données (HP, nom, puissance d'attaque…). Dans ce module, on y ajoute des méthodes : des fonctions déclarées à l'intérieur de la struct, qui agissent sur ses membres.

C'est le pont entre « j'ai des variables groupées » et le futur challenge David & Goliath : deux combattants, chacun avec sa struct, qui s'attaquent tour par tour — d'abord en console, puis affichés dans Raylib.

Ce que l'élève saura faire à la fin

  • Déclarer une struct avec des champs et des méthodes membres
  • Appeler une méthode sur une instance : david.Attack(goliath)
  • Afficher l'état du duel en console et dans la fenêtre Raylib
Prérequis : Print, Variables (struct simple), Conditions, Boucles, STD_Fonctions (fonction + retour struct).

Théorie

Struct + données

On regroupe ce qui décrit un combattant :

struct Fighter
{
    std::string name;
    int hp;
    int attackPower;
};

Méthode membre : Attack

La méthode lit les données de cette instance (l'attaquant) et modifie la cible passée en référence :

void Attack(Fighter& target)
{
    target.hp -= attackPower;
    if (target.hp < 0) target.hp = 0;
    printf("%s attaque %s pour %d degats !\n",
           name.c_str(), target.name.c_str(), attackPower);
}

Autres méthodes utiles

bool IsAlive() const { return hp > 0; }

void PrintStatus() const
{
    printf("%s — HP: %d | ATK: %d\n", name.c_str(), hp, attackPower);
}
⚠️ Pas encore de la POO classique : pas de class, pas d'héritage. On reste sur une struct C++ enrichie — suffisant pour le duel David & Goliath.

Exercices FlashDev

Exercice 1 — Deux combattants

  1. Créez struct Fighter avec name, hp, attackPower
  2. Instanciez david et goliath avec des valeurs cohérentes
  3. Affichez leurs stats dans la console

Exercice 2 — Méthodes et échange

  1. Ajoutez Attack, IsAlive, PrintStatus
  2. Depuis main() : david.Attack(goliath) puis goliath.Attack(david)
  3. Vérifiez IsAlive() pour chacun

Challenge

  • Afficher les HP restants dans Raylib avec DrawText + TextFormat
  • Préparer une boucle while (david.IsAlive() && goliath.IsAlive()) — un tour par combattant
Super challenge (aperçu David & Goliath) : enchaîner un mini combat complet en console, puis refléter l'état du duel à l'écran à chaque tour.
