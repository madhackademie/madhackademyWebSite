Objectif du Module

Nous allons voir rapidement ce qu'est une variable en C++, et surtout les formes les plus importantes (voire indispensables selon les 80/20 de la loi de Pareto).

Nous verrons ultérieurement les variables les plus importantes de la librairie Raylib (dans l'univers 2D selon le même principe).

Nous apprendrons comment créer une variable et y écrire.

Pour commencer, une variable correspond à un espace mémoire réservé dans l'ordinateur. Cet espace est dédié uniquement à cette variable.

Cependant, vous ne pouvez pas stocker n'importe quoi n'importe où : si l'ordinateur attend des bananes, vous ne pouvez pas lui donner des patates. C'est le principe du typage en C++. (bien qu'il effectue parfois des conversions automatiques non souhaitées). La programmation n'étant qu'une suite de traitements et de transformations de données, les variables sont donc indispensables.

On peut distinguer deux grandes familles :
  • les simples (elles stockent une donnée unique)
  • les complexes (elles regroupent plusieurs données ou suivent un modèle)

Commençons par énumérer les types fondamentaux. Le format générique est le suivant :

typeVariable nomVariable ⇒ int age, float tailleEnMètre, bool estVivant

Ce que l'élève saura faire à la fin

Déclarer et typer correctement une variable, utiliser des opérateurs arithmétiques et d'affectation, comprendre le concept de mémoire et les pointeurs basiques.

Théorie

Le Typage

Variables simples :

  • Int ⇒ entier (1, 2, 3, 4, ...)
  • Float ⇒ nombre décimal (1.5)
  • Char ⇒ caractère (a, b, c)
  • Bool ⇒ vrai/faux

Variables complexes :

  1. array - Tableau
  2. struct - Structure

Variables les plus utilisées

IMAGE : Exemples de variables dans l'éditeur
  1. int → un entier, un chiffre rond.
  2. float → un nombre à virgule. Il faut terminer la valeur par la lettre f.
  3. bool → une valeur logique : true ou false, pas d'autre option.
  4. char → un caractère simple (lettre, chiffre, symbole), écrit entre ' '.
  5. Tableau (array) → plusieurs valeurs du même type regroupées.
  6. struct (structure) → un type modulaire qui peut contenir plusieurs valeurs de types différents, représentant une entité (personnage, livre, voiture, etc.).
  7. Pointeur → une variable contenant une adresse (ici un int) initialisée à nullptr(ou null).
  8. Adresse → l'adresse d'une variable existante (ici aussi un int).

Opérateurs

Opérateurs Arithmétiques

Permettent de faire des calculs (résultat : true ou false).

Opérateur Signification Exemple
+ Addition a + b
- Soustraction a - b
* Multiplication a * b
/ Division a / b
% Modulo a % b
++ Incrémenter a++ (a+1)
-- Décrémenter a-- (a-1)

Opérateurs d'Affectation

Permettent de modifier une variable.

Opérateur Signification Exemple
= a = 10 a vaut 10
+= a += 5 a = a + 5
-= a -= 3 a = a - 3
*= a *= 4 a = a * 4
/= a /= 2 a = a / 2

Opérateurs de Comparaison

Permettent de comparer deux valeurs (résultat : true ou false).

Opérateur Signification Exemple
== Égal à a == b
!= Différent de a != b
<= Inférieur ou égal à a <= b
>= Supérieur ou égal à a >= b
< Strictement inférieur a < b
> Strictement supérieur a > b

Opérateurs Logiques

Permettent de combiner les opérateurs de comparaison.

Opérateur Signification Exemple
&& ET a > b && a < c
(a compris entre b et c)
|| OU a > b || c < d

Résumé

Points Clés

  • Point clé 1 : Les variables stockent des données typées en mémoire.
  • Point clé 2 : Types simples : int, float, char, bool ; complexes : array, struct.
  • Point clé 3 : Opérateurs pour calculs, affectations et comparaisons.

À retenir

  • Règle importante : Toujours initialiser les variables pour éviter les bugs.
  • Piège à éviter : Conversions implicites qui masquent des erreurs de typage.
  • Bonne pratique : Utiliser des noms de variables descriptifs comme tailleEnMètre.

Définitions

Cast : Un cast (ou transtypage) est l'opération qui permet de convertir une valeur d'un type de donnée en un autre type. (explication et utilisation ultérieurement dans la formation)

Hexadécimal : Système de numération en base 16 utilisé en programmation pour représenter des adresses mémoire.

Pointeur : Un pointeur est une variable dont le contenu est une adresse.

Console : Élément de l'éditeur qui essaie de communiquer avec nous tant bien que mal.
