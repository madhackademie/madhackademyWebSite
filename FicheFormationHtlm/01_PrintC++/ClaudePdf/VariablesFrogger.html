Introduction

Nous allons voir rapidement ce qu'est une variable en C++, et surtout les formes les plus importantes (voire indispensables).

Nous verrons ultérieurement les variables les plus importantes de la librairie Raylib (dans l'univers 2D).

Nous apprendrons comment créer une variable, y écrire (la lecture se fera à la carte suivante).

Pour commencer, une variable correspond à un espace mémoire réservé dans l'ordinateur. Cet espace est dédié uniquement à cette variable.

Cependant, vous ne pouvez pas stocker n'importe quoi n'importe où : si l'ordinateur attend des bananes, vous ne pouvez pas lui donner des patates. C'est le principe du typage en C++. (bien qu'il effectue parfois des conversions automatiques non souhaitées). La programmation n'étant qu'une suite de traitements et de transformations de données, les variables sont donc indispensables.

On peut distinguer deux grandes familles :
  • les simples (elles stockent une donnée unique)
  • les complexes (elles regroupent plusieurs données ou suivent un modèle)

Commençons par énumérer les types fondamentaux. Le format générique est le suivant :

typeVariable nomVariable ⇒ int age, float tailleEnMètre, bool estVivant

  1. int → un entier, un chiffre rond.
  2. float → un nombre à virgule flottante. Il faut terminer la valeur par la lettre f.
  3. bool → une valeur logique : true ou false, pas d'autre option.
  4. char → un caractère simple (lettre, chiffre, symbole), écrit entre ' '.
  5. Tableau (array) → plusieurs valeurs du même type regroupées.
  6. struct (structure) → un type modulaire qui peut contenir plusieurs valeurs de types différents, représentant une entité (personnage, livre, voiture, etc).
  7. Pointeur → une variable contenant une adresse (ici un int) initialisée à null.
  8. Adresse → l'adresse d'une variable existante (ici aussi un int).

Opérateurs d'Affectation

Permettent de modifier la valeur d'une variable sans tout réécrire.

Opérateur Exemple Correspond à
= a = 10 Met 10 dans a
+= a += 5 a = a + 5
-= a -= 3 a = a - 3
*= a *= 4 a = a * 4
/= a /= 2 a = a / 2

Et chaque case mémoire dispose d'une adresse (attention c'est là que ça devient fumant). Nous allons voir qu'il est possible de modifier une variable via des opérateurs classiques et faciles à prendre en main (voir très intuitifs façon calculatrice). Mais le principe de l'adresse est quelque chose utilisé par l'ordinateur. Et en tant que programmeur C++ tout puissant, nous disposons aussi de cet accès (rapide / direct / empreinte mémoire faible). Il s'agit des pointeurs. On va les survoler très rapidement, on en aura besoin bientôt. Mais pas de panique, vous devez juste savoir ce que c'est et revenir à ce chapitre quand vous y êtes confronté. On les manipulera avec plus de précision et expliquerons leurs intérêts plus tard.

Dans la carte sur le printf() il y avait un %p il sert à afficher l'adresse de la dite case. Et il s'agit d'hexadécimal. Là encore, pas de panique, c'est un code informatique presque matériel. Et si on regarde cette place de parking vue d'en haut, cela correspond à des cellules d'un tableau. C'est pourquoi l'adresse pour simplifier est ici notée N et +1 pour chaque case (cellule) mémoire adjacente. Bien, voyons comment initialiser un pointeur.

int* est une variable qui peut stocker l'adresse d'un int. Il s'agit de l'adresse GPS où est garée la voiture (le chiffre).

& Cible l'adresse de la voiture. Donc avec int* vous créez une variable de type adresse pour un int et avec & vous lui assignez l'adresse d'une variable de type int. On dit donc que int* ⇒ pointe vers &. Si vous devez créer un pointeur vide, il est IMPÉRATIF de le rendre null sinon vous risquez de tout péter.

On peut vraiment citer "De grands pouvoirs impliquent de grandes responsabilités" dans ce cas précis.

Opérateurs de Comparaison

Permettent de comparer deux valeurs (résultat : true ou false).

Opérateur Signification Exemple
== Égal à a == b
!= Différent de a != b
<= Inférieur ou égal à a <= b
>= Supérieur ou égal à a >= b
< Strictement inférieur a < b
> Strictement supérieur a > b

Opérateurs Logiques

Permettent de combiner les opérateurs de comparaison.

Opérateur Signification Exemple
&& ET a > b && a < c
(a compris entre b et c)
|| OU a > b || c < d
